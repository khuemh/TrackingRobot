README

** NOTE:
- On PC/Laptop and Raspi:   . Linux environment
                            . Python3 installed
                            . OpenCV 3.4 installed
                            . sudo modprobe bcm2835-v4l2 (enable Camera)

** Object tracking Python
- Run file Test07.py by using command: python3 Test07.py
- Start tracking by choosing tracked Object by Mouse

** Object color tracking C++
- Build file hsv.cpp by using command: ./cvbuild.sh hsv.cpp. This will create a exe file name output
- Run exe file by using command: ./output
- Select color tracking by adjusting the trackbar HSV
